Setting Global Defaults

By default checkbox.me will set certain config options with a default value. There may be a time where you would like a differnt value to be set. Global options can be set using the options js method. Just set a var with our options and passed it in when you call init


    var options = {
      styles: {
        color: 'red'
      }
    }

    checkbox.init('.checkbox', options);

The above code will take set all of the checkboxes on your page to have a color of red unless they are overrided by the data-style attribute
